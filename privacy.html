Privacy Policy — gog CLI

Last updated: May 4, 2026

gog is a personal command-line tool created and used exclusively by Peter Kim (peter.sy.kim@gmail.com) on his personal Mac. It is not distributed to the public, not available in any app store, and has no other users.

Data collected

gog requests access to the following Google account data:

  • Gmail (read-only): Email message metadata and content, used solely to extract daily briefing information for personal use.
  • Google Calendar (read/write): Calendar events, used to read scheduling context and write briefing-related notes back to the calendar.

How data is used

All data accessed through Google APIs is used exclusively to generate a private daily briefing for Peter Kim. No data is displayed to any third party, stored in any database outside the local machine, or transmitted to any server other than Google's APIs directly.

Data storage

gog processes data in memory on Peter Kim's Mac. No Google user data is persisted to disk beyond what the local OS may temporarily cache. No cloud storage, no database, no logging service.

Data sharing

gog does not share, sell, transfer, or disclose any Google user data to any third party under any circumstances.

Third-party services

gog communicates only with Google's official APIs (googleapis.com). No third-party analytics, advertising, or data services are used.

Security

OAuth tokens are stored in the local filesystem, accessible only to the user account running gog. No tokens are transmitted to or stored on remote servers.
